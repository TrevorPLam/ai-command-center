## MANUAL STATE TRANSITIONS

To manually update state:
1. Update `state` field (pending → in_progress → complete)
2. Update `in_progress_task` when starting work
3. Update `completed_tasks` list when tasks finish
4. Update `next_task` to point to the next task ID
5. Update `last_updated` to current ISO timestamp
6. Save the file
