## GLOBAL CONTRACTS

```yaml
global:
  - id: "$g1"
    version: "1.0.0"
    rule: "All files must use UTF-8 encoding with LF line endings"
    keyword: "encoding"
    anti_pattern:
      severity: B
      description: "Files with CRLF line endings or non-UTF-8 encoding"
      auto_fix: "Convert line endings to LF, ensure UTF-8 encoding"

  - id: "$g2"
    version: "1.0.0"
    rule: "All TypeScript files must have no any types in production code"
    keyword: "typescript"
    anti_pattern:
      severity: S
      description: "Using any type instead of proper type annotations"
      auto_fix: "Replace any with specific types or unknown"

  - id: "$g3"
    version: "1.0.0"
    rule: "All React components must be exported as default or named exports"
    keyword: "react"
    anti_pattern:
      severity: B
      description: "Components without proper exports"
      auto_fix: "Add default or named export statement"

  - id: "$g4"
    version: "1.0.0"
    rule: "All API calls must use TanStack Query for data fetching"
    keyword: "api"
    anti_pattern:
      severity: S
      description: "Direct fetch/axios calls without TanStack Query"
      auto_fix: "Wrap in useQuery or useMutation hook"

  - id: "$g5"
    version: "1.0.0"
    rule: "All state must be in Zustand stores for UI-only state"
    keyword: "state"
    anti_pattern:
      severity: B
      description: "Using React useState for complex UI state"
      auto_fix: "Move to Zustand store"

  - id: "$g6"
    version: "1.0.0"
    rule: >
      Workspace Merkle root for attestations must be computed using the
      canonical algorithm: files sorted lexicographically by UNIX path,
      leaf hash = SHA256 of file content, directory hash = SHA256 of
      concatenated list (child name, type 'file' or 'dir', hash).
      Symlinks are followed and treated as regular files.
    keyword: "merkle"
    anti_pattern:
      severity: S
      description: "Ad‑hoc or inconsistent Merkle tree computation"
      auto_fix: "Use the provided canonical Merkle script"
```
